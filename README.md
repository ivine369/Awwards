# Awwards
An application will allow a user to post a project he/she has created and get it reviewed by his/her peers.
### 9th September 2019

## By Ivine Kemuma

### Description
An application will allow a user to post a project he/she has created and get it reviewed by his/her peers.

### Setup/Installation Requirements

. git clone 
. cd awwards
. pip install -r requirements.txt
. python manage.py runserver
